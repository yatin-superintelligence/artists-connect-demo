# Artist Circle

Welcome to **Artist Circle**! This is a high-fidelity, interactive prototype of a dating and networking application built exclusively for Artists and Creative Professionals.

It is currently deployed and live at:
🌍 [https://artistscircle.yatintaneja.in/](https://artistscircle.yatintaneja.in/)

## 🎨 Overview
Artist Circle provides a sleek, dark-mode focused aesthetic tailored for visual creatives, musicians, designers, and more. Connect, collaborate, and meet other artists around you. 

## ✨ Features
- **Discover**: Swipe and discover other artists and groups nearby. Filter by distance, age, and specific artistic disciplines.
- **Likes & Matches**: See who has liked you and match with mutual connections.
- **Rich Chat interface**: Real-time mock chat featuring read receipts, typing indicators, replies, and reactions.
- **Voice & Video Calls**: In-app calling interface with picture-in-picture floating widgets, mute, and video toggles.
- **Profile Management**: Customize your bio, add your artistic interests, and manage your portfolio images.
- **Pro Features (Demo)**: Experience premium features like *Incognito Mode*, sending *Sparks* to stand out, and *Uplifting* your profile to be seen by more people.
- **Customizable Themes**: Change the visual vibe with built-in theme management.

## 🚀 Tech Stack
This application is built using modern frontend technologies:
- **React (v19)**: Component-driven UI.
- **TypeScript**: For robust, strongly-typed code.
- **Vite**: Ultra-fast frontend tooling and building.
- **Tailwind CSS / Custom CSS**: For styling and the sleek dark aesthetic.

## 🛠 Running Locally

To run this application locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yatin-superintelligence/artists-connect-demo.git
   cd artists-connect-demo
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
